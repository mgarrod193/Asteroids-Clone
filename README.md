# Asteroids Clone (Godot)

This is a small Asteroids-style game I built in Godot as a learning project. The goal was to recreate the classic arcade feel while practicing physics-based movement, game loops, visual effects, audio implementation, and integrating an online leaderboard; while also becoming more familiar with the Godot engine.

You rotate, thrust, shoot, and try not to slam into space rocks.

---

## Features

- **Wave-based progression** – Each wave increases the difficulty with more and faster asteroids.  
- **Asteroid splitting** – Large asteroids break into smaller pieces when destroyed.  
- **Physics-based movement** – Inertia-driven controls inspired by the original arcade feel.  
- **Particle effects** – Asteroids burst into particles on destruction.  
- **Sound effects and music** – Includes gameplay SFX and background music.  
- **Damage feedback** – When the player takes damage, they lose a life, their position resets, and the ship flashes briefly to provide clear visual feedback.
- **Online leaderboard** – High scores are saved so you can compete for the top spot.  

---

## What I Practiced / Learned

- Designing wave spawning and difficulty scaling using resources
- Handling collision detection and object splitting  
- Adding particle effects and audio for better game feel  
- Connecting a game to an online leaderboard  
- Completing and polishing a small but finished project
